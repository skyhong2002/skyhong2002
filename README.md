### Sky Hong 洪軾凱

於21世紀初自天空降落，

患有先天性反社會人格障礙精神病，

自稱以尋找人生目標為藉口，

實則奉命研究潛在命運石之門的真理。


專長撇開音樂和打code，

簡單來說就是個獨流的毒瘤。
